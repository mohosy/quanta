# Quanta

A browser-based N-body physics playground that compares brute-force and spatial-grid acceleration strategies in real time.

## Highlights
- Live switch between `O(n^2)` brute force and grid-accelerated neighborhood search.
- Particle count and gravity controls for stress-testing performance.
- Runtime metrics for frame time, interaction estimates, average speed, and energy.
- Fully client-side and runnable by opening `index.html`.

## Stack
- HTML5 Canvas
- Vanilla JavaScript
- CSS dashboard UI

## Run
Open `index.html` in your browser.
